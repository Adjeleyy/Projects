2. What is an html attribute?

An html attribute is a code that gives more information about a specific html element.

3. What is the purpose of the alt attribute in an img tag?

The alt attribute in an img tag serves as an alternative text in place of an image on a webpage if the image cannot be loaded due to poor connection or an error in the code.

4. What is the difference between an absolute url and a relative url?

An absolute url defines the location of a file starting with the root folder to the file itself. A relative url on the other hand specifies the location of a file from the location of the code. (it defines the location of a file relative to the code).

5. List 7 html tags (h1 - h6 not included)

a tag, p tag, img tag, br tag, body tag, title tag, html tag

6. What is an empty html element? Give 2 examples

An empty html element is one has no closing tag. Examples are the br tag and img tag.

8. What is the current, most recent version of html?

9. What is the root html element of every html page?

html (with the angle brackets) is the root html element of every html page.

10. What is the difference between viewing the html source, and inspecting an html element?

Viewing the html source shows the user the original html code that was used to build the page, while inspecting an html element includes any changes that have been made to the html file.
